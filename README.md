smart-make
==========

Reusable GNU/Make scripts (same as smart-build but much simpler).

Examples
========

  * [Build a C program](examples/build-c-program)

Variables
=========

Module Target Name
------------------
  * PROGRAM
  * LIBRARY

Source Files and Sub-modules
----------------------------
  * SOURCES
  * SRCDIR

Compile Flags
-------------
  * CFLAGS
  * CXXFLAGS

Link Flags
----------
  * LDFLAGS
  * LOADLIBS
